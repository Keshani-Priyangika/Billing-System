# K & K Fancy Shop - Billing System

A modern, desktop-based Billing and Invoice Generation System developed using Python and Tkinter. This application features a fully customized, borderless user interface with custom control buttons (Minimize, Maximize, Close) and interactive window dragging capabilities.

## Features
- **Custom Modern UI:** Borderless window design with a sleek, clean color scheme.
- **Dynamic Cart System:** Easily select items from the shop menu, input quantities, and add them to the cart with instant total amount calculation.
- **Real-Time Input Validation:** Prevents errors by checking for valid input quantities.
- **Invoice Preview:** Generates a structured, neat text-based invoice inside the application.
- **Clear All Data:** Reset the system instantly to process the next customer.
